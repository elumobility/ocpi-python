# CRUD Operations

Learn how to implement CRUD operations for OCPI modules.

## Abstract Implementation

The `Crud` class provides the abstract interface for all CRUD operations in OCPI Python.

::: ocpi.core.crud.Crud
    options:
      show_source: true
      members:
